Git Commands 

git clean -f ==> Remove Untracked Files 
git commit --ammend --not edit ==> Add File to commit 
git reset --hard origin/<branch> ==> Reset Local Head to Branch Head
git checkout MERGE_HEAD - <PATH/TO/FILE> ==> Take Master Version in merge conflict 
git checkout Head <PATH/TO/FILE> ==> Reset File
git reset --> UN Add Staged Files
